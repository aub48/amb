# Integrated Real Time Communication & Notification System

এই ZIP ফাইলটি GitHub Pages-ready static website হিসেবে তৈরি করা হয়েছে।

## ফাইলসমূহ
- `index.html`
- `style.css`
- `script.js`
- `assets/ambulance-bg.svg`

## GitHub Pages-এ চালানোর নিয়ম
1. ZIP ফাইলটি unzip করুন।
2. ভিতরের সব ফাইল GitHub repository-এর root folder-এ upload করুন।
3. Repository Settings → Pages → Branch `main` এবং folder `/root` select করুন।
4. Save করার পর GitHub Pages URL থেকে ওয়েবসাইট দেখা যাবে।

> শুধু ZIP ফাইল repository-তে upload করলে GitHub Pages ওয়েবসাইট হিসেবে ZIP-এর ভিতরের `index.html` চালাবে না। ফাইলগুলো unzip করে root-এ রাখতে হবে।
